## 📌 项目简介

### 创意背景

> Yesterday is history, tomorrow is a mystery, but today is a gift. That is why it is called the present.<br>
> _Kung Fu Panda_

Present 是一款面向年轻群体的同频的、真实的社交应用。“Present” 既指正在发生的“当下”，亦指值得珍藏的“礼物”，传达“当下的每一真实的瞬间，均是社交生活赠予的礼物”的产品理念。
