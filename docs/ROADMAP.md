# Fragment 路线图

## V0.3 本地稳定性基础（当前）

- [x] Model / Data / Service / ViewModel / Theme / Utils 分层
- [x] ArkData RDB、版本迁移和空库种子
- [x] 稳定 ID 与展示编号分离
- [x] 社区私密副本、来源记录和重复收存拦截
- [x] 编辑、归档、恢复、永久删除和关系清理
- [x] 四级可见性、授权记录、共享撤回和首次隐私说明
- [x] 回响年龄、窗口、冷却、每日上限、打开与稍后状态
- [x] Canvas 空值保护、扩大命中区和文字关系列表
- [x] Hypium 单元测试和真实 ArkTS/HAP 构建
- [x] phone-only 产品声明

## V0.4 系统能力

- [ ] Notification Kit 调度应用外回响
- [ ] Form Kit 桌面卡片
- [ ] 图片、录音和视频碎片
- [ ] 车机与快捷入口
- [ ] Tablet / 2in1 响应式布局与真机验收
- [ ] 分布式数据库和跨设备同步

## V0.5 社区服务

- [ ] 账号和最小化身份系统
- [ ] 真实社区查询、内容审核、举报和删除
- [ ] 指定联系人授权选择器
- [ ] 服务端幂等键与离线重试
- [ ] 数据导出、账号注销与保留策略

## V0.6 智能关联

- [ ] 端侧 embedding 和向量索引
- [ ] 相似、相反、延续、启发分类
- [ ] 可解释关联原因与置信度
- [ ] 用户确认后才持久化建议关系

AI 仅作为后续可替换的关系建议模块；当前版本不调用真实模型，也不把演示树描述为 AI 结果。
