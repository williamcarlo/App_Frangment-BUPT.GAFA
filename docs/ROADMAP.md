# Fragment 迭代路线

## V0.1 基础骨架

- [x] Stage 模型工程
- [x] 碎片数据模型
- [x] 内存仓库与示例数据
- [x] 标签关系服务
- [x] Box、捕捉面板与基础延申页

## V0.2 三屏交互闭环

- [x] Box → 个人轨道 → 社区延申三屏手势导航；
- [x] 关键词、Tag 与类型搜索；
- [x] 个人关系图节点返回具体碎片；
- [x] 社区树按相似、相反、行动逐层展开；
- [x] 社区节点收存与继续写闭环；
- [x] 碎片内容与共享范围必填校验；
- [x] 应用内按通勤、睡前、周末窗口筛选低频回响；
- [x] 新增与收存碎片动态加入个人脑洞网；
- [x] 私密碎片禁止作为社区延申起点；
- [ ] ArkData Preferences 或 RDB 持久化；
- [ ] 碎片编辑、删除与归档；
- [ ] 通过系统通知调度应用外的未来回响；
- [x] 统一时间戳，并按本地日历边界显示今天、昨天与日期；
- [ ] 首次使用引导与隐私说明。

## V0.3 HarmonyOS 特性

- Form Kit 桌面回响卡片；
- Notification Kit 未来提醒；
- 车机语音快速输入；
- 图片、录音和视频碎片；
- 手机记录、平板网络、PC 生长的多端布局；
- 手表短语音捕捉；
- 同应用跨设备数据同步。

## V0.4 智能关联

- 端侧文本 embedding；
- ArkData 向量数据库；
- 相似、相反、延续与启发关系分类；
- 显示关联原因与置信度；
- 用户确认后才写入正式关系。

## V0.5 社区接枝

- 逐条公开授权；
- 从自己的碎片出发检索社区；
- 接枝而非点赞；
- 社区内容审核、举报与撤回；
- 每日有限探索入口，避免形成无限流。
