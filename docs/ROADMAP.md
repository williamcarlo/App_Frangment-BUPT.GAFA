# Fragment 迭代路线

## V0.1 基础骨架

- [x] Stage 模型工程
- [x] 碎片数据模型
- [x] 内存仓库与示例数据
- [x] 标签关系服务
- [x] 首页、捕捉面板与延申网络页

## V0.2 本地可用

- ArkData Preferences 或 RDB 持久化；
- 碎片编辑、删除与归档；
- 模糊时间偏好和提醒状态；
- 搜索与 tag 筛选；
- 首次使用引导与隐私说明。

## V0.3 HarmonyOS 特性

- Form Kit 桌面回响卡片；
- Notification Kit 未来提醒；
- 手机记录、平板网络、PC 生长的多端布局；
- 手表短语音捕捉；
- 同应用跨设备数据同步。

## V0.4 智能关联

- 端侧文本 embedding；
- ArkData 向量数据库；
- 相似、相反、延续与启发关系分类；
- 显示关联原因与置信度；
- 用户确认后才写入正式关系。

## V0.5 社区接枝

- 逐条公开授权；
- 从自己的碎片出发检索社区；
- 接枝而非点赞；
- 社区内容审核、举报与撤回；
- 每日有限探索入口，避免形成无限流。
