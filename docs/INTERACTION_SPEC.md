# Fragment 交互规格

## Box

- 点击主视觉打开输入 Sheet；
- 内容必填，最多 500 字；Tag 为空时使用“未命名碎片”；
- 回响偏好可选通勤、睡前、周末、先让它沉睡、随机；
- 可见范围默认私密，用户必须明确选择后才能保存；
- 首次启动展示隐私说明；
- 当日回响可以打开，也可以“今天先不看”。

## 个人碎片

- 横向滑动切换一级页面；顶层使用并行手势，避免被垂直 List/Scroll 吞掉；
- 垂直浏览时，选中状态保存稳定碎片 ID，不保存数组下标；
- 搜索匹配展示编号、Tag 和内容；
- 选中卡片可编辑、归档；社区可见碎片还可撤回共享；
- 归档箱支持恢复和永久删除，破坏性动作都要二次确认；
- 关系图使用 Canvas，同时提供文字关系列表和更大的节点命中范围。

## 社区树

- 只能从 `COMMUNITY` 且未归档的个人碎片进入；
- 私密中心会自动切换到第一条合格公开碎片；没有合格项则拒绝进入；
- 点击末端节点按本地规则展开分支；
- “继续延申”携带当前 Tag，但可见范围仍需重新明确选择；
- “收进 Box”先检查稳定来源 ID；成功后生成私密副本、写入关系和保存记录，然后退出社区并定位新副本；
- 返回个人页时清空社区选中路径，下一次进入重新从公开根开始。

## 回响

- 年龄至少 12 小时；
- 归档和 `NONE` 永不进入候选；
- 通勤、睡前、周末按本地时间窗；随机仅在白天；
- 同一碎片冷却 7 天；
- 每个本地日最多 surface 一张；
- 记录 `surfacedAt`、`openedAt`、`dismissedUntil` 和展示原因。

## 未实现

系统卡片、系统通知、车机语音、媒体碎片、真实社区接口、审核举报、指定联系人选择器、跨设备同步和 AI 关联尚未实现。
